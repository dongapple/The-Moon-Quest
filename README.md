# theMoonQuest

## 설명

별소녀 리메이크
달소녀 프로젝트

## 커밋 컨벤션

### 형식:

<이모지> <설명> [#이슈번호]

### 이모지와 타입:

✨ feat: 새로운 기능 및 코드 추가
🔥 remove: 기능 및 코드 제거
🌊 modify: 기능 및 코드 수정
🎨 style: 코드 스타일링
➕ add: 파일 추가
➖ remove: 파일 제거
📦 move: 파일 이동
🏗️ structure: 프로젝트 구조 변경
🐛 fix: 버그 수정
🚑 hotfix: 치명적인 버그 수정
♻️ refactor: 코드 리팩토링
📝 doc: 문서 작성 및 편집
🎉 init: 초기 커밋
🚀 deploy: 프로젝트 배포
